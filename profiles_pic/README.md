# Profile Pictures Directory

This directory stores profile pictures for face recognition.

## How to Add Your Photo

1. Take a clear, front-facing photo of yourself
2. Rename the file to your first name (e.g., `John.jpg`, `Alice.png`)
3. Place it in this directory
4. The system will automatically load it

## Supported Formats

- .jpg / .jpeg
- .png
- .gif
- .bmp

## Tips for Best Results

- Use well-lit photos
- Front-facing angle works best
- One face per photo
- Clear, high-resolution images

## Example

If your name is "Sarah", save your photo as:
- `Sarah.jpg` or
- `Sarah.png`

The filename (without extension) will be used as your name when the system greets you.
